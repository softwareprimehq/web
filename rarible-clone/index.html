EIP 1967 introduces a standardized method for storing proxy information, streamlining the process by which block explorers access and present this data to users. The EIP 2981 standard empowers NFT-equivalent contracts to use the ERC-1155 and ERC-721 interfaces, allowing them to define the royalty fee that NFT creators are required to make whenever their NFT is sold. The initiative is designed to streamline royalty fees across the NFT marketplace, ensuring that all participants in the ecosystem can support these fees globally.
