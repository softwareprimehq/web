PopTok - User App

Discover All the Features of the User App Here

Signup - User

Users can easily sign up with PopTok using email, phone, or other social media accounts. The process includes OTP verification, profile setup, and selecting their preferred video category and language for seamless engagement.

Messaging - User

The messaging feature allows users to communicate with any other users via text form. They can also engage in real-time conversations and enhance social interactions within the app.

Like and Comments - User

Users can show their appreciation with likes, and leave comments on streaming videos in the PopTok app. It fosters community engagement through feedback, discussions, and social connections.

Change Password - User

The Change Password feature allows users to update their password securely. Users can enter their existing password, create a new one, and confirm it to improve account security and prevent unauthorized access.

Favorite Video - User

Users can save videos, sounds, and hashtags to their favorites in the PopTok app. This feature allows them to quickly access their favored content, ensuring a seamless discovery and better engagement.

Notification Settings - User

Users can manage push notifications for likes, comments, follows, and messages by enabling or disabling them in settings. It provides a customized setup with minimal interruptions.

Free coins for ad - User

PopTok allows users to earn free coins by watching ads and adding them to their wallets. This feature improves user engagement with in-app activities, app revenue, and platform retention.

Report Video - User

Users can report inappropriate videos with valid reasons and descriptions provided by the admin. This feature helps maintain a secure streaming environment by ensuring all content aligns with community guidelines and platform standards.

Live Streaming - User

Creators can go live on the platform and set an entry fee, allowing users to pay for access to exclusive live streams. This enables creators to earn more virtual coins from their content. It provides an engaging experience to their audiences.

Coin Purchase - User

Users can securely purchase coins to watch the streaming videos from creators they do not follow. Multiple payment options ensure seamless transactions and a hassle-free purchasing experience.

Blocking Users

This feature allows users to restrict interactions from specific accounts. It prevents them from viewing profiles, sending messages, or accessing videos, providing a safer and more personalized experience.

Report User

PopTok allows users to report other users by selecting their accounts and providing a valid reason using the report option. This will maintain an effective video streaming experience for all users.

Video Upload - In-app Audio

Users can effortlessly select, edit, and post videos with categories, descriptions, tags, and comments, along with in-app audio options for enhanced creativity. They can expand their reach by paying to post promotional videos.

PopTok - Admin

A Powerful Dashboard to Control & Optimize Your Platform

Manage admin

The admin can create new admin users by providing their name, and email, and assigning specific roles to easily manage various activities inside the PopTok video streaming app.

Manage audio

Admin can manage audio by adding, editing, organizing, and removing tracks. They can upload new audio files, alter existing ones, properly classify them, and ensure smooth playback for users.

Promoted video management ( Promoting video from admin)

PopTok allows admins to promote videos by setting details like days, age limits, and gender preferences. It enhances targeted audience engagement and maximizes reach for better visibility.

Wallet management

Admin can manage wallets by adding coins for users and ensuring smooth financial operations. This enhances transparency, builds trust, and maintains seamless in-app payments for multiple users.

Site settings

This feature allows the admin to manage overall site settings by entering the platform name, copyright details, view limits for promotion, etc. It improves site visibility and platform growth.

Assigning Roles to Subadmin

Admins will assign roles and permissions to sub-admin for managing and overseeing daily operations. From that, sub-admins can take responsibility for handling user requests, enforcing policies, and ensuring smooth workflow.
